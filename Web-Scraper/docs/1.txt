Способы обхода блокировки ботов:
1. Использование IP ротации (отправка запросов через прокси)
2. Изменение геолокации для получения данных для конкретной локализации (отправка запросов через прокси)
3. Регулирование частоты запросов к сайту
4. Имитация человека (частота обращений к сайту, неоднотипные запросы, ошибочные действия, смена юзерагентов)
5. Решение каптчи!
6. Сокрытие запросов на получение конкретных данных (почты, номера телефонов, адреса)
